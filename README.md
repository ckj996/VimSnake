# VimSnake

> Snake game with Vim key binding

## Compile

    make vimsnake

## Run

    ./vimsnake [milisecond]

## Control

    'hjkl' for LEFT DOWN UP RIGHT
    'q' to quit

## Depend on

* ncurses
