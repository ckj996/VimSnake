# VimSnake

> Snake game with Vim key binding

## Compile

    make vimsnake

## Run

    ./vimsnake [milisecond]

## Control

    'h': LEFT
    'j': DOWN
    'k': UP
    'l': RIGHT
    'q': QUIT
    'r': RESTART

## Depend on

* ncurses
